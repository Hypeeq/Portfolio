Information about the Employer

During my co-op term, I had the privilege of working at the University of Guelph, specifically within the School of Computer Science. The University of Guelph is renowned for its focus on research and innovation. My work was closely aligned with the field of educational technology, where I helped develop an interactive, compiler-embedded e-textbook to support computer science students.

Interesting fact: The University of Guelph campus is a hub for innovative research in various disciplines, and it attracts students from across the globe to its diverse and inclusive community.

Goals

For this work term, I set several key learning outcomes:

  • Job Task Goals: My primary goal was to review and enhance the current integration of compilers within an existing e-textbook. This included creating an interactive interface to allow students to code without leaving the textbook.
  • Skills Development: I aimed to deepen my knowledge of front-end development, specifically using ReactJS and JavaScript, while improving communication skills through user manual documentation.
  • Technology Exposure: My goal was to gain hands-on experience with JavaScript frameworks, UI/UX design, and API integration. I also wanted to explore embedding development tools into educational platforms, which would benefit my future work in educational technology.

In reflecting on these goals, I successfully improved my skills in UI development and API integration, while also learning more about independent learning and teamwork.

Job Description

My role at the University of Guelph focused on designing and developing a user interface for a code execution environment embedded within an e-textbook. The key task was to create a seamless experience for students to write and test code directly within the textbook, without the need for external tools. This project required:

  • Knowledge of frontend development using React
  • Backend API development using Node.js
  • Understanding of code compilation and execution processes

The most exciting aspect of my job was learning to work with compilers in a web-based setting, which is a unique and challenging task that blends several areas of computing science.

Conclusions

In conclusion, this project allowed me to enhance my skills in both frontend and backend development while working on a unique problem: embedding a code compiler within an interactive educational platform. If someone were to describe the contents of this portfolio, I hope they would highlight:

  • My ability to solve complex problems
  • My expertise in both web development and system integration
  • My commitment to creating solutions that improve learning experiences
